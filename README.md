# ⚡ My Finance Dash

**Your Finance. Your Data. 100% Local.**

My Finance Dash is a pro-grade, local-first financial dashboard built into a **single, portable HTML file**. It provides advanced analytics, Sankey flow diagrams, and spending heatmaps without ever sending your data to a cloud server.

## 🛡️ The "Local-First" Philosophy

In a world where financial tools monetize your data, My Finance Dash takes a different approach: **Zero Servers. Zero Tracking.**

* **100% Client-Side:** All processing happens in your browser's RAM.
* **No Databases:** Your bank data never leaves your machine.
* **Volatile Memory:** Once you close the tab, your data is gone.
* **Ultimate Portability:** It's just one HTML file. Keep it on a USB drive, run it offline, use it anywhere.

## ✨ Key Features

* **Sankey Flow Diagrams:** Visually track how your money flows from income sources to spending categories.
* **Spending Heatmaps:** Instantly identify your most active financial days with an interactive calendar.
* **Smart KPIs:** Real-time updates on your Savings Rate, Net Profit, and Total Balance.
* **Subscription Detector:** Automatically flags recurring payments so you never lose track of fixed expenses.
* **Instant Export:** Generate a high-quality PNG report of your dashboard with a single click.
* **Dark Perfected UI:** A modern, professional interface with glassmorphism effects, optimized for readability.

## 🚀 How to Use It

It couldn't be simpler:

1.  **Download:** Grab the latest `index.html` from the https://maxciber.github.io/myfinancedash page.
2.  **Open:** Double-click the file to open it in your favorite web browser (no internet connection required).
3.  **Drag & Drop:** Export your transactions from your bank (Excel/CSV) and drag the file into the dashboard.
4.  **Analyze:** Explore your financial data instantly.

## 🛠️ Tech Stack

This project proves you don't need complex infrastructure for high-end analytics. Built entirely with:

* **Vanilla JavaScript** (No heavy frameworks)
* **TailwindCSS** (via CDN for styling)
* **Chart.js & Google Charts** (For data visualization)
* **SheetJS** (For local Excel/CSV parsing)
